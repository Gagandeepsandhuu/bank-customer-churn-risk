# Bank Customer Churn Risk Prediction

## 📌 Project Overview
Bank Customer Churn Prediction is a Machine Learning project that predicts whether a customer is likely to leave the bank or continue using its services. The goal of this project is to help banks identify high-risk customers and take preventive actions to improve customer retention.

## 🎯 Objectives
- Analyze customer behavior and identify churn patterns.
- Perform data preprocessing and feature engineering.
- Train and evaluate multiple Machine Learning classification models.
- Predict customer churn risk.
- Deploy the trained model using Streamlit for real-time predictions.
